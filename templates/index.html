What is AudioMasher?

  • Live, in-browser audio and music programming environment
  • Based on the amazing Sporth audio programming language by Paul Batchelor
  • Source available on Github
  • Created by Pierre Cusa (software | music)

Where do I start?

  • Browse and listen to some existing patches
  • Go through the interactive tutorial

What is the license on scripts posted to AudioMasher?

A license (or the URL to a license) can be placed in the patch description or in a comment in the code itself. If there is no license there, all patches and the resulting audio are by default protected under the folliwing license:

Attribution-NonCommercial-ShareAlike 3.0 Unported .
